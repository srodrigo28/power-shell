# Sequencia de Treinamento Rapido (PowerShell Basico)

## Objetivo
Em 30 a 45 minutos, aprender os fundamentos para criar e executar scripts simples.

## Ordem sugerida
1. 01-ola-mundo.ps1
2. 02-variaveis.ps1
3. 03-condicionais.ps1
4. 04-loops.ps1
5. 05-funcoes.ps1

## Roteiro de aula
1. Executar o script 01 e validar ambiente.
2. Entender entrada de dados com Read-Host no script 02.
3. Praticar tomada de decisao com if/elseif/else no script 03.
4. Repeticao com for e foreach no script 04.
5. Criar e chamar funcao com parametros no script 05.

## Exercicios rapidos
1. Alterar o script 02 para pedir cidade.
2. No script 03, incluir regra para numero maior que 100.
3. No script 04, listar 5 ferramentas que voce usa no dia a dia.
4. No script 05, criar funcao Multiplicar.
